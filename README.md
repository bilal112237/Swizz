Point of Sale (POS) System 🛒
Name: Swizz
Course: Object Oriented Programming (OOP)
Project Type: Java Swing Application with MySQL Database

📖 Project Overview
This is a complete Point of Sale (POS) software designed to help businesses manage inventory and process sales. The application features two main interfaces: an Admin Dashboard for managing products and viewing reports, and a Cashier Dashboard for processing transactions.

We built this project using Java Swing for the visual interface and MySQL for the database, following strict OOP principles like Encapsulation, Inheritance, and Design Patterns.

🚀 Key Features
1. Admin Module
   Login Security: Secure login system that detects if the user is an Admin or Cashier.

Inventory Management: Add new products, update prices, and delete items.

Sales Reports: View transaction history to see what has been sold.

2. Cashier Module
   User-Friendly Interface: Modern grid layout with buttons for selecting products.

Dynamic Cart: Add items, view sub-totals, and calculate the final bill automatically.

Stock Control: The system prevents selling items that are out of stock.

Real-time Updates: Updates the database immediately when a sale is finished.
